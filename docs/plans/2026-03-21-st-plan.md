## 3. 入口/出口标准

**入口条件：**
- 所有功能状态为passing
- 环境已准备
- 所有必需配置存在

**出口条件：**
- 所有回归/集成/E2E测试通过
- 所有NFR阈值满足
- 无Critical或Major缺陷

## 4. 风险优先级

1. **关键路径** - 核心用户工作流（知识库创建→文档上传→问答）
2. **集成边界** - 跨功能数据流（文档→解析→向量→检索→LLM）
3. **NFR阈值** - 性能、错误处理
4. **边缘情况** - 错误恢复、无内容检索
5. **兼容性** - 浏览器差异

## 5. 测试环境

- 后端: http://localhost:8080
- 前端: http://localhost:5174
- 数据库: MySQL (内嵌H2用于测试)
- 向量数据库: Chroma (本地文件)
- LLM: MiniMax API (已配置)
