## 1. 视觉风格方向

### 风格：简洁现代 (Clean Modern)

**风格描述**: 白色/浅灰色背景，蓝色主色调，圆角卡片，现代简洁。参考豆包、Cursor等AI聊天工具的界面风格，强调内容优先、交互流畅。

**色彩倾向**:
- 主色调：蓝色系 (#3B82F6)
- 背景：浅灰色 (#F9FAFB) + 白色 (#FFFFFF)
- 强调：柔和的阴影和圆角

**字体方向**:
- 无衬线字体，现代感
- 高可读性，层次分明

**布局方向**:
- 左侧边栏 + 右侧主内容区
- 卡片式设计，内容区域分明
- 响应式布局，适配不同屏幕

**目标用户匹配**: 企业内部员工，普通用户，无需专业技术背景

## 3. 组件提示

### 3.1 按钮

#### 主要按钮 (Primary Button)
**SRS Trace**: FR-010

**基础提示**:
> 圆角矩形按钮，高度40px，宽度自适应内容+24px padding，背景色 #3B82F6，白色14px中等字重文字，圆角6px，无边框。按钮下方有0 4px 6px -1px rgba(0,0,0,0.1) 阴影。悬停时背景变为 #2563EB，按钮稍微上移2px，阴影加深。点击时背景变为 #1D4ED8。

**悬停态**: 背景 #2563EB，transform: translateY(-2px)，阴影加深  
**激活态**: 背景 #1D4ED8，transform: translateY(0)  
**禁用态**: 背景 #DBEAFE，文字 #9CA3AF，cursor: not-allowed

## 4. 页面提示

### 4.1 主聊天页面

**SRS Trace**: FR-008, FR-009, FR-010

**布局描述**:
> 全屏布局：左侧240px固定侧边栏 + 右侧自适应主区域。侧边栏包含顶部"新建知识库"按钮 + 知识库列表。主区域顶部40px高度显示当前知识库名称 + 设置按钮，底部80px固定高度输入区域，中间为消息列表区域（可滚动）。

**完整页面提示**:
> 页面使用 Inter 字体，#F9FAFB 页面背景。左侧240px侧边栏：顶部"新建知识库"按钮 #3B82F6 蓝色，下方知识库列表每项48px高，悬停 #F3F4F6，选中 #DBEAFE + 左侧蓝边。右侧主区域：顶部横条显示当前知识库名称，右侧设置图标。中间消息区域白色背景，用户消息右对齐蓝色气泡，AI消息左对齐白色带灰边框气泡。底部输入框区域白色背景，上边框#E5E7EB，左侧发送按钮蓝色圆形，右侧附件图标。整体阴影柔和，圆角现代。

**关键交互**:
- 点击知识库：切换到该知识库的对话，加载历史消息
- 发送消息：Enter键或点击发送按钮，输入框清空，AI消息流式输出
- 新建知识库：弹出模态框，输入名称后创建

**响应式行为**:
- **桌面 (>= 1024px)**: 左侧240px侧边栏 + 右侧内容区
- **平板 (768-1023px)**: 侧边栏可收起，点击汉堡菜单显示
- **移动 (< 768px)**: 侧边栏全屏显示，点击知识库后全屏聊天

## 5. 样式规则与约束

### 5.1 辅助功能

- 所有交互元素焦点可见（outline: 2px solid #3B82F6）
- 按钮最小点击区域 36px × 36px
- 颜色对比度符合 WCAG AA 标准
- 表单错误实时提示

### 5.2 动画

- 过渡时间：150ms - 200ms
- 缓动函数：ease-in-out
- 消息出现：淡入 + 轻微上移
- 按钮悬停：轻微上移 + 阴影加深

### 5.3 响应式断点

| 断点 | 宽度 |
|------|------|
| 移动 | < 768px |
| 平板 | 768px - 1023px |
| 桌面 | >= 1024px |

### 5.4 命名规范

- CSS类名使用 BEM 或 CSS Modules
- 组件名使用 PascalCase
- 颜色、间距等使用 CSS 变量
